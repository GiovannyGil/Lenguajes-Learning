# OPA App Movil Backend

## Installation Dev

install dependencies npm
```bash
npm install
```

- set vars in *config.json* based in *config.example.json*
- copy file *example.env* to *.env* and replace vars
- create folders: ssl, temp


## Build

install pkg global
```bash
npm install -g pkg
```

build exe server for windows
```bash
npm run package:win
```
